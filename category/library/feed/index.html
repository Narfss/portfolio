TransformableDrawableButton es una librería para android. Con ella se pueden crear botones con símbolos comunes y que el cambio entre ellos sea fluido.


Se pueden crear iconos con símbolos como cruces, aspas, flechas entre otros. La librería esta preparada para crear estos con un fondo circular, aunque de igual manera se puede decorar el botón con un fondo personalizado.
No obstante se puede tener un texto y los símbolos se pueden colocar a cualquiera de los lados del botón.



La librería esta disponible para su su integración con Gradle. En cuanto a la configuración se realiza por medio de parámetros en el layout, lo que hace que sea sencillo y libre de código extra.

ParallaxEverywhere es una librería para Android. Con esta librería es fácil conseguir un efecto sobre cualquier imagen o texto al desplazar un contenido. Ya sea listas, scrolls o grids, incluso sobre algunos elementos móviles.



La integración es con Gradle y su configuración es sencillamente en el layout, no siendo necesaria ninguna implementación de código.

Entre los parámetros de configuración se encuentran el sentido, el bloqueo o la transición de el mismo.
